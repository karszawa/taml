# taml
日毎の時間割を表示するiOSアプリです。
科目ごとの欠席数を簡単に記録できるので出席日数をキッチリ管理したい人におすすめです。
数ある時間割アプリの中でも最も直感的で小綺麗なUIを実現できたと思います。
